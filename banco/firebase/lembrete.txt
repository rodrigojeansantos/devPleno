			*********** Entendendo ***********
É um dataBase As a Service (Baas)
É otimo para chats
RealTimeDataBase é no formato Json
Ele é em tempo real.

Real time DataBase: Para banco mais simples
CloudFireStore: Bd mais complexos.
Autenticação: Para login e criação de usuário
Analicts: Analise de usuário e aplicações devices.
CloudFunctions: Aplicações Serverlees
Testes: Para performance
CloudMessage: Push notification
Integração sobre produtos da google.

			*********** Metodos ***********
ref(''): Cria dados na url informada.
child: Acessa o nó filho passado por parametro.
push: Diferente do set, ja insere os dados e coloca o id baseado em data.
once(): retorna os dados lidos de uma url
snapshot: Objeto retornado pela leitura
